Welcome to the  fisheryresources-web webservice application

This is the a REST based webservice using this xml schema. Go here to for more documentation.


URL Resource Description
                                                                                                                                                                                                                                     
stocksbyspecies/fao3alphacode/{fao3alphacode} StocksBySpecies Service FisheryResources with the operation StocksBySpecies. It searches through all resource factsheets given a species(fao3alphacode) and returns a list of stocks. 
                                                                                                                                                                                                                                     
stocksbyfaoarea/faoarea/{faoarea} StocksByFaoArea Service FisheryResources with the operation StocksByFaoArea. It searches all stocks ocurring in the given FAO area (or sub-area, division, sub-division) or intersecting with it. Results are presented in the elements Inclusion and Intersection.
